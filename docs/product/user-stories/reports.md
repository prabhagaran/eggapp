# User Stories — Reports

**Priority:** P2 (001–002) / P3 (003–005) · **Surface:** Web

## US-RPT-001: Hatch performance [Web, P2]
As the Owner, I want per-batch fertility / hatch-of-set / hatch-of-fertile and the trend across batches (filterable by species, incubator, date range), so I can see if my process is improving. Rules: BR-015

## US-RPT-002: Environmental history per batch [Web, P2]
As the Owner, I want a per-batch environment report (min/max/avg, excursion list with durations) so I can correlate a bad hatch with conditions.

## US-RPT-003: Vaccination compliance [Web, P3]
As the Owner, I want compliance across flocks (administered/due/overdue counts, full traceable history per flock).

## US-RPT-004: Mortality trends [Web, P3]
As the Owner, I want mortality by flock/stage/cause over time, benchmarked against domain-knowledge norms (§5).

## US-RPT-005: CSV export [Web, P3]
As the Owner, I want any report exportable as CSV.
- Given any report view, when I export, then the CSV matches the on-screen filters exactly.
