# Glossary

This glossary defines key terms and abbreviations used throughout the  
**Reusable Environmental Control Platform** documentation.

---

## A

**Actuator**  
A device that physically affects the environment, such as a heater, fan, pump, or motor.

**Alarm**  
A system condition indicating abnormal or unsafe operation that requires user attention or action.

---

## C

**Control FSM**  
A Finite State Machine responsible for making decisions based on sensor data and system state.

---

## D

**DS18B20**  
A digital temperature sensor using the 1-Wire protocol, known for accuracy and noise immunity.

**DHT22**  
A digital humidity and temperature sensor commonly used in environmental monitoring.

---

## E

**Encoder (Rotary Encoder)**  
An input device that detects rotational movement and button presses, used for UI navigation and value adjustment.

**Event Group**  
A FreeRTOS mechanism used for signaling events between tasks.

---

## F

**FSM (Finite State Machine)**  
A computational model consisting of states, transitions, and actions used to implement deterministic behavior.

**FreeRTOS**  
A real-time operating system used to manage tasks, timing, and inter-task communication.

---

## H

**HAL (Hardware Abstraction Layer)**  
A software layer that isolates hardware-specific details from higher-level logic.

**Humidity Control**  
The process of maintaining a target humidity level using sensors and actuators.

---

## I

**I²C (Inter-Integrated Circuit)**  
A serial communication protocol used for connecting peripherals such as OLED displays.

---

## L

**Logging**  
The process of recording system data and events for monitoring and diagnostics.

---

## M

**MVP (Minimum Viable Product)**  
The first complete and functional version of the system that validates the architecture and core features.

---

## P

**Profile**  
A configuration that defines system behavior, enabled features, and limits without changing control logic.

---

## R

**Relay**  
An electrically operated switch used to control high-power loads safely.

**RTOS (Real-Time Operating System)**  
Software that manages task scheduling and timing in real-time systems.

---

## S

**Sensor**  
A device used to measure environmental parameters such as temperature, humidity, or water level.

**Setpoint**  
The target value that the control system attempts to maintain.

**System FSM**  
The top-level FSM that manages overall system operation modes.

---

## T

**Task**  
An independent execution unit managed by FreeRTOS.

**Temperature Control**  
The process of regulating temperature using heating and cooling actuators.

---

## U

**UI (User Interface)**  
The means by which a user interacts with the system, consisting of an OLED display and rotary encoder.

---

## W

**Water Level Control**  
Monitoring and maintaining water availability for system operation.

---

## Summary

This glossary provides consistent terminology across documentation and implementation.
